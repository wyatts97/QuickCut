# FFmpeg Directory

This directory contains FFmpeg binaries for video processing.

For development purposes, you can download FFmpeg binaries from:
- Windows: https://ffmpeg.org/download.html#build-windows
- macOS: https://ffmpeg.org/download.html#build-mac
- Linux: https://ffmpeg.org/download.html#build-linux

Place the ffmpeg.exe, ffplay.exe, and ffprobe.exe files in this directory for Windows builds.
